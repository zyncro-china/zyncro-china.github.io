rubymotion for android

等了很久的rubymotion for android终于可以用了。按照官方网站升级:

  1. update rubymotion
  2. install android sdk
  3. install android ndk
  4. uncompress sdk and ndk
  5. set environment variable
  6. motion create –template=android AndroidHello
  7. rake device 最后一步出现错误：
ERROR! It looks like your version of the NDK does not support API level L.

寻找原因:

先运行android升级到最新的sdk, 最后在这里看到如何设置 api version

Motion::Project::App.setup do |app|
  app.name = "MyAndroidMotionApp"
  app.api_version = "19"
end
