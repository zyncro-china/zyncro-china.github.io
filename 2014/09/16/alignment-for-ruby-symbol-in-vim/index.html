alignment for ruby symbol in vim


在编辑器里做代码对齐

为了提高阅读性， 应尽量使其美观，对齐是其中一种方式。ruby中的symbol使用广泛，为了对其他们，需要使用一个vim插件Tabular

symbol in ruby1.8

使用下面key mapping，可以对其ruby1.8的symbol

vmap <leader>a> :Tabularize /:/r1c0l0<CR>:Tabularize /=>/l1c1l0<CR>

#before alignment
mymethod arg1, :a => 1,
  :c => 3,
  :sdfds => 23,
  :fsd => 2

#after alignment
mymethod arg1, :a     => 1,
               :c     => 3,
               :sdfds => 23,
               :fsd   => 2

symbol in ruby1.9

使用下面key mapping，可以对其ruby1.9以后的symbol

vmap <leader>a: :Tabularize /:/r0c1l0<CR>

#before alignment
mymethod arg1, a: 21,
  bd: 234,
  ccccc: 43434

#after alignment
mymethod arg1, a: 21,
              bd: 234,
           ccccc: 43434
